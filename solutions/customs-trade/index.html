The Problem

Customs documentation is broken at scale.

Brokers, freight forwarders, and importers run on a flood of invoices, BOLs, packing lists, and certificates of origin — in every format, language, and layout. Tariff volatility and tightening sanctions enforcement mean every classification error is now expensive.

3–7d

Customs Holds Per Error

A single documentation inconsistency — mismatched values, missing COO, wrong HS code — triggers customs holds that delay shipments by 3 to 7 days.

2–4hr

Per Entry, Manually

Coordinators read 4–6 documents per shipment, re-key 50+ fields into customs systems, look up HS codes, and check sanctions lists by hand — on every single entry.

10K+

Documents Per Branch / Month

A single forwarder branch processes tens of thousands of shipping documents per month. Shared inboxes overflow. Back-office cost spirals against thin margins.

The Solution

Agentic AI that understands trade documents.

Xentovia's AI agents read every customs document — multi-language, handwritten, scanned, EDI — extract structured data, classify HS codes, screen for sanctions, and prepare audit-ready filings. Human reviewers see only the 10–20% that need judgment.

Customs Entry Preparation

Auto-draft customs declarations from invoice, BOL, and COO data. Pre-fill 50+ fields, attach rationale, and route to a reviewer with one click.

HS Code Classification

Suggest HS codes with confidence scores. Agents learn from each broker's historical classifications to sharpen accuracy over time and reduce duty leakage.

Sanctions & Trade Screening

Continuous checks against OFAC, EAR, ITAR, EU dual-use, and global sanctions lists. Restricted parties caught before filing — not after a hold.

Cross-Document Validation

Compare values, parties, and quantities across invoice, BOL, packing list, and COO. Inconsistencies flagged pre-submission — no more customs holds for mismatched data.

HITL Review Workflow

Low-confidence outputs route to your reviewer with side-by-side document viewer, inline edit, and one-click approve. Reviewers handle only the 10–20% that need judgment.

Audit-Ready Trail

Every agent decision logged with reasoning, source documents, and reviewer actions. Audit-ready trail for any regulator, any time — without a manual file pull.

Document Types

Licenses & Permits
